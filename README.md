# E-Library Management System

A simple **Console-Based Library Management System** written in C. This project helps users manage book records with various features like adding, searching, and deleting books.

---

## 📖 Project Overview

**E-Library** is a user-friendly command-line application for managing a collection of books. It includes a login system and multiple book management options.

### Features

- 🔐 **Secure Login** (3 attempt limit)
- 📚 **Add New Book**
- 🔍 **Search Book by Name**
- 🔍 **Search Book by Author**
- 📋 **Show All Books**
- 🗑️ **Delete Last Book**
- 🗑️ **Delete Any Book** (by index)
- 💰 **Budget Friendly Search** (±50 TK range)
- 🎨 Clean decorative interface with borders
- In-memory storage (data resets when program closes)

---

## 🛠 Technologies Used

- **Language**: C
- **Compiler**: GCC (Recommended)
- **Libraries**: `stdio.h`, `string.h`, `stdbool.h`, `stdlib.h`
